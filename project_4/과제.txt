과제 1: 회원가입 페이지 만들기
:렌치: 필수 구성 요소
이름, 이메일, 비밀번호, 비밀번호 확인 입력란
성별 선택 (라디오 버튼)
관심사 선택 (체크박스 여러 개)
자기소개 (textarea)
가입하기 버튼

:예술: 스타일 요구사항
입력창과 라벨을 나란히 또는 줄맞춤해서 정리
form을 중앙 정렬
position 속성을 사용해 아래 중 하나 이상 구현:
오른쪽 아래 고정된 “도움말” 버튼 (position: fixed)
배경에 이미지를 넣고 position: fixed로 전체 고정


과제 2: 로그인 페이지 만들기
:렌치: 필수 구성 요소
이메일, 비밀번호 입력란
로그인 버튼
아이디 기억하기 체크박스
비밀번호를 잊으셨나요? 링크


:예술: 스타일 요구사항
로그인 폼을 페이지 중앙에 배치
비밀번호를 잊으셨나요? 문구를 폼 안의 오른쪽 하단에 absolute로 배치
배경에 이미지를 넣고 position: fixed로 전체 고정
위 과제 제출은 index.html에 메뉴에 링크를 추가하고 도메인만 링크로 제출해주세요.
